# 💫 About Me:
Mi vida giró en torno a las Artes Visuales y el poder de la fotografía. <br>A través de este camino, desarrollé una perspectiva única para <br>observar, analizar y resolver problemas de manera creativa.<br>Mi mente disfruta de la libertad del arte y también se siente atraída <br>por la estructura y la lógica de la tecnología. Hoy, he decidido unir estos <br>dos mundos aparentemente opuestos traduciendo la creatividad de <br>mis años como artista a la lógica del código.<br><br>De artista visual a desarrolladora Fullstack. <br>Hoy, no solo capturo la realidad, la estoy programando.
